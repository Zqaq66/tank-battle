# 🎮 坦克大战 (Tank Battle)

一个经典的坦克大战网页游戏，使用 HTML5 Canvas 和原生 JavaScript 开发。

## 🕹️ 游戏特色

- 🎯 经典的坦克大战玩法
- 🧱 可破坏的砖墙和坚固的钢铁墙
- 🤖 智能敌方坦克 AI（会追踪玩家）
- 📊 3 个精心设计的地图关卡，循环递增难度
- 💥 粒子爆炸特效
- 🛡️ 玩家复活无敌保护
- 📱 支持移动端触屏操作
- 🎨 精美的像素风格画面
- 🏆 分数和击杀统计

## 🎮 操作说明

| 按键 | 操作 |
|------|------|
| `↑` `↓` `←` `→` | 移动坦克 |
| `W` `A` `S` `D` | 移动坦克（备选） |
| `空格` | 发射子弹 |
| `R` | 重新开始游戏 |

### 移动端

- **左侧屏幕**: 滑动控制移动方向
- **右侧屏幕**: 点击发射子弹

## 🚀 快速开始

直接在浏览器中打开 `index.html` 文件即可开始游戏！

```bash
# 或者使用任意 HTTP 服务器
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## 🎯 游戏规则

1. 控制你的绿色坦克消灭所有敌方坦克
2. 每关有 20 辆敌方坦克，消灭全部即可过关
3. 砖墙可以被子弹摧毁，钢铁墙需要更高火力
4. 被敌方子弹击中会损失一条命
5. 失去所有生命则游戏结束

## 🛠️ 技术栈

- 原生 JavaScript (ES6+)
- CSS3 动画与布局
